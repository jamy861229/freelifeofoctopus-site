# USD Stock Tool（美股匯率試算器）

## 工具定位

快速試算：

👉 美股報酬 + 匯率變動 → 實際報酬

解決問題：

- 股票賺錢但匯率吃掉收益
- 不知道要漲多少才回本
- 不知道匯差影響

---

## 核心功能

### 模式一（價格模式）

輸入：

- 買入價格
- 賣出價格

輸出：

- 報酬率 %

---

### 模式二（報酬模式）

輸入：

- 預期報酬 %

輸出：

- 目標賣出價格

---

### 匯率輸入

- 匯率買入（TWD/USD）
- 匯率賣出（TWD/USD）

---

## 結果區（重點）

必須顯示：

- 匯差損失率
- 股票保本報酬率
- 股票安全報酬率
- 預期報酬

---

## 狀態提示

使用 3 狀態：

- 🟢 安全（賺）
- 🟡 打平 / 接近
- 🔴 虧損
